Screen pages (in order)

The screen updates immediately after saving — no restart needed. Up to 8 stops; the button on the device cycles through them in this order.

Night schedule

While asleep the screen is dark and polling pauses; a button press wakes it and cycles pages. Changes apply immediately. Times are local (Toronto).

Network

The device restarts and joins this network. If it can't connect it brings the TTC-Setup hotspot back so you can try again. Once connected, this page lives at http://ttc.local.
